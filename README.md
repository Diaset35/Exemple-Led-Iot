# Exemple Led Iot
![arduino](https://sdz-upload.s3.amazonaws.com/prod/upload/arduinoBouton.png)
Ce projet permet d'allumer une Led à distance depuis internet.
## Bibliothéque utilisée

+ Wifi
+ Adafruit-MQTT
+ NodeRED
# Matériel

+ **ESP32**
+ Serveur MQTT
+ Led
+ Fils de connexion
+ Alimentation microUSB +5v
